
# Mon Portfolio (en React)

## Description

Ce projet est mon portfolio personnel, développé avec React et TypeScript. Il est conçu pour montrer mes compétences et projets.

## Structure du projet

Le projet utilise Vite pour une configuration minimale avec React et TypeScript. Il inclut des règles ESLint pour assurer la qualité du code.

## Installation

Pour installer et configurer ce projet localement, suivez ces étapes :

1. Clonez le dépôt :
   ```bash
   git clone https://github.com/Thibolognaize/Portfolio-2025.git
   ```
2. Installez les dépendances :
   ```bash 
   npm i
   ```
3. Démarrer localement :
   ```bash 
   npm run dev
   ```
4. (optionnel) Build : 
   ```bash 
    npm run build
   ```
